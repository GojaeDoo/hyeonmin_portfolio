## 프로젝트 정보
- **프로젝트명**: 최현민 포트폴리오 웹사이트
- **개발 기간**: 2025.06 ~ 2025.07
- **기술 스택**: React, TypeScript, Next.js, Styled-Components
- **배포**: Vercel에 배포

## 아키텍처
- **Next.js 13+ App Router** 사용
- **Container-Presenter 패턴** 적용
- **TypeScript**로 타입 안정성 확보
- **Styled-Components**로 스타일링

실행 방법:

```bash
yarn dev
```
